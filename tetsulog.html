FEATURES

鉄道趣味のすべてが、
ひとつのアプリに。

乗り鉄、撮り鉄、廃線鉄、形式鉄。すべての「鉄」が、自分の趣味を最高品質で記録できる。

FEATURE 01 · 図鑑

編成番号で集める、
「鉄道版ポケモン図鑑」。

E235系トウ47編成、N700S系J36編成。形式じゃなく、「個体」を記録する。 遭遇した編成が増えるほど、図鑑が埋まっていく。

  • JR・私鉄・地下鉄の全形式に対応
  • 編成番号・車番・所属まで個体識別
  • 廃車間近の編成は赤色アラート表示
  • 「未撮影編成」リストでコレクション欲を駆動
FEATURE 02 · OCR

写真を撮るだけで、
編成番号が読み取られる。

iOS純正のLive Text技術で、車体の編成番号をその場で認識。 手入力ゼロで、撮影と同時に図鑑に記録される。完全に端末内処理、外部に送信なし。

  • VisionKitによる端末内OCR（API料金ゼロ）
  • 誤読時はワンタップで候補修正
  • 写真EXIFから撮影日時・位置を自動取得
  • 暗所・斜め撮影でも高精度
FEATURE 03 · MAP

あなたの足跡を、
地図にプロットする。

乗った区間が緑のラインで塗られていく。撮影した地点には色分けされたピン。 「日本全国どこまで行ったか」が、ひと目でわかる。

  • MapKitによる高精度な日本全国地図
  • 乗車区間の自動ハイライト
  • 撮影地ピンに写真・編成・日時を紐付け
  • 路線別・形式別・年別の絞り込み
FEATURE 04 · 光線計算

順光か、逆光か。
瞬時に判定する。

撮影地と日時を入れるだけで、その瞬間の太陽方位・順光逆光・影の長さを計算。 「あの場所、何時に行けばベストか」をアプリが教えてくれる。

  • 緯度経度＋日時で太陽方位を精密計算
  • 順光・逆光・斜光の3段階判定
  • 朝・昼・夕方のシミュレーション
  • 列車進行方向と太陽方向のクロス分析
FEATURE 05 · タイムライン

あなたの鉄道人生を、
年表にする。

蓄積した記録から、年・月別のタイムラインが自動生成。 「2018年5月、あなたは189系に乗りました。この形式は2019年に引退しました。」

  • 記録から自動でタイムライン生成
  • 引退・廃車情報と自動リンク
  • 年間まとめを動画として書き出し可能
  • 家族・次世代への継承メディアとして

FEATURE 07 · LIVE ACTIVITY

乗車中は、ロック画面が
「運転台」になる。

乗車を開始すると、Dynamic Island とロック画面にライブアクティビティが出現。 今乗っている編成・通過した駅・次の停車駅・経過時間が、画面を開かなくてもリアルタイムで更新される。 すべて端末内処理、外部通信ゼロ。

乗車セッション

ポケットの中で、
旅が進んでいく。

記録した編成で乗車を始めれば、あとはポケットに入れたまま。 Dynamic Island をちらっと見るだけで、次の駅も、乗車時間も、走行距離もわかる。 降りたら自動でその区間が記録に残る。

  • Dynamic Island に今乗っている編成番号を常時表示
  • ロック画面で通過駅・次駅・経過時間をライブ更新
  • 降車時に乗車区間・距離を自動で図鑑へ記録
  • ActivityKit による端末内駆動・通信ゼロ・電池に優しい
  • Apple Watch のスマートスタックにも同時表示

FEATURE 08 · 接近アラート

狙いの編成を、
逃さないために。

会いたい形式・編成を「ウォッチリスト」に登録。その車両が走る路線エリアや、登録した撮影地に近づくと、 内蔵ダイヤをもとに「そろそろ来るかも」とアラート。GPSとダイヤだけで動くので、通信もサーバーも不要です。

ウォッチリスト

「会いたい車両」を
登録しておくだけ。

引退間近の189系、まだ撮れていない編成、推しの形式。 ウォッチリストに入れておけば、あなたが該当エリアに足を踏み入れた瞬間、 ダイヤと照らし合わせて静かに教えてくれる。

  • 形式・編成番号単位でウォッチリスト登録
  • 登録撮影地・主要駅へのジオフェンス接近通知
  • 内蔵ダイヤから「通過しそうな時間帯」を端末内で推定
  • 未捕獲編成・廃車予定編成を優先的にハイライト
  • 位置情報の利用は端末内のみ、どこにも送信されません
※ リアルタイムの在線位置ではなく、あなたの位置とダイヤ・運用傾向から「会える可能性」を予測する機能です。確実な遭遇を保証するものではありません。

FEATURE 10 · 音鉄

走行音も、駅メロも、
あの瞬間の音を、残す。

VVVF音、ジョイント音、駅メロディ、車内放送。 遭遇記録に音を添えれば、写真では伝わらない「その瞬間」が永久保存されます。完全に端末内、外部送信はゼロ。

音鉄レコーダー

音は、写真より
記憶を呼び戻す。

撮るだけじゃなく、聴いて記録する。発車メロディ、加速時のVVVF、終着のジョイント音。 引退してしまった形式の最後の音まで、あなたの図鑑に紐づけて残せます。

  • 遭遇記録に音声を複数本添付可能
  • レベルメータ付きの録音UI（AAC m4a・端末内保存）
  • 編成詳細から再生・進捗バー表示
  • マイク権限・音声データはすべてあなたの端末内のみ
  • JASRAC等の権利配慮で共有機能は持たず、純粋に個人の記録用途

WHY WE BUILT THIS
あの車両に、
「最後にもう一度」は、
二度と来ないかもしれない。

引退する形式、消えていく路線、塗り替えられる塗装。 鉄道趣味は、いつも「もう会えないかもしれない」と隣り合わせ。 だからこそ、撮った一枚を、乗った一区間を、見送った最後の日を、 ちゃんと残しておきたい。TetsuLog は、あなたの鉄道人生の記録係です。 キャラクターを集めるためでも、誰かに見せるためでもなく、 ただ、あなた自身のために。
