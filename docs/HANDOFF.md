## ✅ 完成しているもの

### プロダクト企画
- App Store鉄道アプリ市場の全領域調査（`docs/market-analysis.md`）
- 「キャラを買うんじゃなく、自分の鉄道人生を記録する」コンセプト確立
- 3次にわたる辛口レビューと是正（`docs/critical-review*.md`）

### iOSアプリ（実装）
- **記録**: 遭遇・乗車を編成番号/号機/車番レベルで／編集・削除・検索・グルーピング
- **図鑑**: 形式別コレクション率／コンプリート／廃車進行表示
- **OCR(Pro)**: VisionKitで編成番号を端末内読み取り
- **順光計算(Pro)**: NOAA算法で順光/逆光判定
- **音鉄録音(Pro)**: 走行音・駅メロを記録に添付
- **地図**: 乗車区間・撮影地・廃線オーバーレイ
- **統計**: 累計距離・Top形式/路線/駅・月別チャート・ラストラン
- **ライブアクティビティ**: 乗車中をDynamic Island/ロック画面に
- **接近リマインダー**: GPS×位置でウォッチ編成を想起
- **撮影地20プリセット**: 記録ゼロでも順光計算が試せる
- **データ自由**: JSON書き出し・読み込み、他サービスからCSV移行
- **Apple Watch**: 今週遭遇数・クイック記録
- **ウィジェット**: ホーム/ロック画面に遭遇数・集めた編成
- **多言語**: 日英中韓

## 🛠 残っているのは「あなたのMacでの作業」だけ

開発（コード・素材）はクラウド上で完了済み。実機リリースは Mac の Xcode が必須で、
全手順は **`docs/release-checklist.md`** に集約してある。要点：

1. **識別子の置換**（フェーズ1）
   `yourname`/`com.example`/`example.github.io`/`support@example.com`/事業者情報 を実値へ
2. **ターゲット追加**（フェーズ2）
   Capabilities（iCloud/App Groups/Push）、Widget/Watch/Tests を追加
3. **課金登録**（フェーズ3）
   App Store Connect で Non-Consumable `...pro` を ¥980 登録、Sandboxテスト
4. **CloudKitスキーマ Deploy**（フェーズ4）
5. **審査素材入力**（フェーズ5・`docs/app-store-assets.md`）
6. **法務ページ公開**（フェーズ6）
7. **ビルド & 提出**（フェーズ7）

> まず触るだけなら `app/README.md` / `docs/run-on-device.md` の最小構成手順で、
> 無料Apple IDでも実機にインストールできる（`.xcodeproj` 同梱済み）。
