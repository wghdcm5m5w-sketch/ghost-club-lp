# TetsuLog 1.1 — What's New

## 日本語

```
v1.1 アップデート

・Siri / ショートカット対応：「テツログで記録」「テツログで乗車開始」と話すだけ
・iPad に最適化：サイドバー＋詳細の2列レイアウト
・音鉄の録音にタグ（走行音／駅メロ／車内放送／警笛／発車ベル）
・写真・録音をまとめて書き出し（端末移行・バックアップに）
・同じ日に同じ編成を記録しようとすると、そっと教えてくれる重複ヒント
・統計の「よく行く駅・路線」が表記ゆれ（JR新宿／新宿駅／新宿）を賢く統合
・狙いの編成の接近アラートを、現在地に近い撮影地から賢く監視
・VoiceOver 読み上げを改善

引き続き、広告なし・追跡なし・サブスクなし。データは100%あなたのiCloudの中。
```

## English

```
What's New in 1.1

- Siri & Shortcuts: just say “Log with TetsuLog” or “Start a ride”
- Optimized for iPad: two-column sidebar + detail layout
- Tags for your train recordings (running sound / station melody / in-car announcement / horn / departure bell)
- Export all photos & recordings at once (for backup and device migration)
- Gentle duplicate hint when logging the same formation twice in a day
- Smarter station/line stats that merge naming variants (JR Shinjuku / Shinjuku Sta. / Shinjuku)
- Approach alerts now monitor the shooting spots nearest to you
- Improved VoiceOver accessibility

Still no ads, no tracking, no subscription. Your data stays 100% in your iCloud.
```

> 注: 録音タグ・iPad2列・正規化・重複ヒント・近傍監視・VoiceOver は本体ビルドに含まれv1.1で出荷可能。
> ウィジェット「廃車進行中」・Watchコンプリは xcodegen フル構成移行後に有効（docs/xcodegen-migration.md）。
